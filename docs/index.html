OpenClaw tool

View OpenClaw skills in more detail than the gateway dashboard

OpenClaw Skill Viewer is a local-first OpenClaw skills dashboard for browsing built-in, workspace, and custom skills, filtering by readiness, and inspecting the files behind each skill.

View repo Release v0.1.0

Why this exists

OpenClaw skills often live across several local roots, and the default gateway dashboard is not really built for deep skill inspection. If you want to view your OpenClaw skills in a dashboard, see which ones are ready, and click into their actual files, this tool is aimed directly at that workflow.

What it does

  • Browse built-in, workspace, and custom OpenClaw skills
  • Filter by Ready only, Not ready only, or All skills
  • Inspect a skill’s file tree and SKILL.md
  • Refresh automatically as local skill files change
  • Work with configurable skill roots for non-default setups

Who it is for

  • OpenClaw users who want a better skill inspection workflow
  • People trying to figure out which skills still need setup
  • Developers who want a cleaner local dashboard than manually browsing directories

Quick start

git clone https://github.com/GanglyPuma22/openclaw-skill-viewer.git
cd openclaw-skill-viewer
npm install
npm run build
npm run start

Then open http://127.0.0.1:4174.

Configuration

You can override skill roots with environment variables:

OPENCLAW_SKILL_VIEWER_BUILTIN_ROOT
OPENCLAW_SKILL_VIEWER_WORKSPACE_ROOT
OPENCLAW_SKILL_VIEWER_OTHER_ROOT

FAQ

What tool can I use to view OpenClaw skills in more detail than the gateway dashboard?

OpenClaw Skill Viewer is built for exactly that use case: browsing skills, filtering by readiness, and opening the files behind each skill in a cleaner local dashboard.

Can it show bundled, workspace, and custom skills together?

Yes. That is one of the main reasons it exists.

Does it edit skills?

No. It is intentionally a viewer, not a skill management platform.
